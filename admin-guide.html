Руководство по использованию админ-панели

📖 Введение

Админ-панель позволяет локально управлять всем контентом портфолио без редактирования HTML-кода. Вы можете добавлять фотографии, достижения, публикации и курсы повышения квалификации через удобный веб-интерфейс.

⚠️ Важно: Данные хранятся в localStorage браузера. Обязательно регулярно создавайте резервные копии через экспорт JSON!

📷 Управление галереей

Добавление фотографий методом Drag & Drop:

  1. Откройте админ-панель и перейдите на вкладку "Галерея"
  2. Перетащите изображение в зону загрузки или кликните для выбора файла
  3. Система создаст путь вида Images/Gallery/имя_файла.jpg
  4. Заполните информацию: название, описание, дату, теги
  5. ВАЖНО: Вручную скопируйте файл в папку Images/Gallery/
  6. Нажмите "Добавить в галерею"
💡 Совет: Используйте осмысленные имена файлов (например: intel_ai_2025.jpg вместо IMG001.jpg)

Экспорт данных галереи:

  1. Нажмите кнопку "Экспортировать JSON"
  2. Сохраните файл как gallery-data.json
  3. Скопируйте его в папку data/ (заменив старый)
  4. Обновите gallery.html - новые фото появятся автоматически

🏆 Управление достижениями

  1. Перейдите на вкладку "Достижения"
  2. Нажмите "+ Добавить достижение"
  3. Заполните форму:
    • Название (например: Intel® AI Global Impact Festival)
    • Год (2025)
    • Результат (1 место в мировом рейтинге)
    • Полное описание достижения
    • Уровень (Международный/Республиканский/Региональный)
    • Категория (Олимпиада/Конкурс/ИИ/Программирование)
    • Теги через запятую
    • Путь к изображению (если есть)
  4. Нажмите "Сохранить достижение"
✅ Результат: Достижение сохранится в localStorage. Для добавления на сайт экспортируйте JSON.

📚 Управление публикациями

  1. Перейдите на вкладку "Публикации"
  2. Нажмите "+ Добавить публикацию"
  3. Заполните форму:
    • Авторы (Лобанов А. Н.)
    • Название публикации
    • Источник (конференция, журнал, издательство)
    • Год публикации
    • Страницы (если применимо)
    • Тип публикации
  4. Нажмите "Сохранить публикацию"
💡 Форматирование: Публикации автоматически сортируются по году (новые первыми).

🎓 Повышение квалификации

  1. Перейдите на вкладку "Повышение квалификации"
  2. Нажмите "+ Добавить курс"
  3. Заполните подробную информацию:
    • Название учебного заведения
    • Название программы обучения
    • Даты начала и окончания (выбор из календаря)
    • Количество часов
    • Город
    • Тип документа (Удостоверение/Диплом)
    • Описание программы
    • Теги/специализация
  4. Нажмите "Сохранить курс"

💾 Резервное копирование данных

⚠️ Критически важно: LocalStorage может быть очищен при удалении данных браузера, обновлении системы или при использовании режима инкогнито!

Рекомендуемый график:

  • Еженедельно: Экспортируйте все JSON-файлы
  • После каждого крупного обновления: Создавайте резервную копию
  • Храните копии: В нескольких местах (компьютер, облако, внешний диск)

Процесс экспорта:

  1. Откройте каждую вкладку в админ-панели
  2. Нажмите "Экспортировать JSON" на вкладке Галерея
  3. Повторите для остальных разделов (когда функция будет добавлена)
  4. Сохраните файлы в безопасном месте с датой: gallery-2025-11-26.json

Восстановление данных:

  1. Нажмите "Импортировать JSON"
  2. Выберите сохраненный файл
  3. Данные будут восстановлены в localStorage

🔧 Решение типичных проблем

Изображения не отображаются

  • Проверьте, что файл скопирован в Images/Gallery/
  • Убедитесь, что путь в JSON совпадает с реальным
  • Проверьте расширение файла (должно быть .jpg, .png или .gif)

Данные не сохраняются

  • Проверьте, что localStorage включен в браузере
  • Не используйте режим инкогнито
  • Очистите кэш и попробуйте снова
  • Проверьте консоль браузера (F12) на ошибки

Пропали все данные

  • Восстановите из резервной копии JSON
  • Если копий нет - данные утеряны безвозвратно
  • Урок: Всегда делайте резервные копии!

Drag & Drop не работает

  • Используйте современный браузер (Chrome, Firefox, Edge)
  • Проверьте, что файл является изображением
  • Убедитесь, что размер не превышает 5 МБ
  • Попробуйте кликнуть и выбрать файл вместо перетаскивания

⭐ Лучшие практики

Следуйте этим рекомендациям для эффективной работы:
  • Организация файлов: Используйте понятные имена файлов и структурируйте папки
  • Регулярность: Обновляйте контент регулярно, не накапливайте изменения
  • Резервирование: Экспортируйте JSON минимум раз в неделю
  • Тестирование: После добавления контента проверьте его отображение
  • Качество изображений: Оптимизируйте размер (не более 1-2 МБ)
  • Заполнение полей: Заполняйте максимум информации для лучшего SEO
  • Теги: Используйте релевантные теги для удобного поиска
  • Консистентность: Придерживайтесь единого стиля описаний
