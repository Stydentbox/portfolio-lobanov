🚀 БЫСТРЫЙ СТАРТ
────────────────────────────────────────────────────────────────

1. Откройте admin.html в браузере
   - Двойной клик по файлу admin.html
   - Или правой кнопкой → Открыть с помощью → Chrome/Firefox

2. Выберите нужную вкладку:
   📷 Галерея
   🏆 Достижения
   📚 Публикации
   🎓 Повышение квалификации

3. Добавьте контент через формы

4. Экспортируйте JSON для резервного копирования


📸 ДОБАВЛЕНИЕ ФОТО В ГАЛЕРЕЮ (Drag & Drop)
────────────────────────────────────────────────────────────────

Шаг 1: Перетащите изображение в зону загрузки
        (или кликните и выберите файл)

Шаг 2: Заполните форму:
        - Название события
        - Описание
        - Дата (ГГГГ-ММ)
        - Теги

Шаг 3: ВАЖНО! Скопируйте файл изображения в папку:
        Images/Gallery/

Шаг 4: Нажмите "Добавить в галерею"

Шаг 5: Для публикации на сайте:
        - Нажмите "Экспортировать JSON"
        - Сохраните как gallery-data.json
        - Скопируйте в папку data/ (заменив старый файл)


🏆 ДОБАВЛЕНИЕ ДОСТИЖЕНИЯ
────────────────────────────────────────────────────────────────

1. Вкладка "Достижения" → "+ Добавить достижение"

2. Заполните:
   - Название: Intel® AI Global Impact Festival
   - Год: 2025
   - Результат: 1 место в мировом рейтинге
   - Описание: Полное описание...
   - Уровень: Международный
   - Категория: Искусственный интеллект
   - Теги: ИИ, международный, победа

3. "Сохранить достижение"


📚 ДОБАВЛЕНИЕ ПУБЛИКАЦИИ
────────────────────────────────────────────────────────────────

1. Вкладка "Публикации" → "+ Добавить публикацию"

2. Заполните:
   - Авторы: Лобанов А. Н.
   - Название: Информационные технологии...
   - Источник: Материалы конференции...
   - Год: 2018
   - Страницы: С. 191
   - Тип: Материалы конференции

3. "Сохранить публикацию"


🎓 ДОБАВЛЕНИЕ КУРСА
────────────────────────────────────────────────────────────────

1. Вкладка "Повышение квалификации" → "+ Добавить курс"

2. Заполните:
   - Учебное заведение: МФТИ
   - Программа: Машинное обучение...
   - Даты: 10.06.2020 - 24.08.2020
   - Часы: 72
   - Город: Москва
   - Тип: Удостоверение
   - Теги: ML, Управление проектами

3. "Сохранить курс"


💾 РЕЗЕРВНОЕ КОПИРОВАНИЕ
────────────────────────────────────────────────────────────────

⚠️  КРИТИЧЕСКИ ВАЖНО! ⚠️

Данные хранятся только в localStorage браузера!
При очистке данных браузера - ВСЁ УДАЛИТСЯ!

ОБЯЗАТЕЛЬНО:
- Экспортируйте JSON каждую неделю
- Храните копии в нескольких местах
- После каждого крупного обновления - экспорт!

Как экспортировать:
1. Кнопка "Экспортировать JSON"
2. Сохранить файл с датой: gallery-2025-11-26.json
3. Хранить в безопасном месте


🔧 ЧАСТЫЕ ПРОБЛЕМЫ
────────────────────────────────────────────────────────────────

❌ Фото не отображается
   ✓ Проверьте: файл скопирован в Images/Gallery/
   ✓ Проверьте: путь совпадает с именем файла
   ✓ Проверьте: расширение .jpg, .png или .gif

❌ Данные не сохраняются
   ✓ Не используйте режим инкогнито
   ✓ Проверьте localStorage в настройках браузера
   ✓ Очистите кэш и попробуйте снова

❌ Пропали все данные
   ✓ Восстановите из резервной копии JSON
   ✓ Если копий нет - данные утеряны!
   ✓ В будущем: регулярный экспорт!

❌ Drag & Drop не работает
   ✓ Используйте Chrome, Firefox или Edge
   ✓ Размер файла не более 5 МБ
   ✓ Попробуйте кликнуть вместо перетаскивания
