# Гайд по управлению галереей

## Доступ к админ-панели

Откройте в браузере: `admin.html`

## Функции админ-панели

### 1. **Просмотр галереи**
При открытии админ-панели автоматически загружаются все изображения из `data/gallery-data.json`. Каждое изображение отображается с полной информацией:
- Миниатюра фотографии
- Путь к файлу
- Название
- Описание
- Дата
- Теги

### 2. **Редактирование изображения**
Для каждого изображения вы можете:
- Изменить путь к файлу (`src`)
- Обновить название
- Отредактировать описание
- Изменить дату (формат: YYYY-MM)
- Добавить или изменить теги (через запятую)
- Нажать кнопку **"Сохранить"** для сохранения изменений

### 3. **Удаление изображения**
- Нажмите кнопку **"Удалить"** под изображением
- Подтвердите удаление в диалоговом окне
- Изображение будет удалено из галереи

### 4. **Добавление нового изображения**
- Нажмите кнопку **"+ Добавить фото"** в верхнем левом углу
- Заполните поля в форме:
  - **Путь к изображению**: путь к файлу (например: `Images/Gallery/photo.jpg`)
  - **Название**: краткое название события/проекта
  - **Описание**: подробное описание
  - **Дата**: в формате YYYY-MM
  - **Теги**: ключевые слова через запятую
- Нажмите **"Добавить"**

### 5. **Экспорт данных**
- Нажмите кнопку **"Экспортировать JSON"**
- Файл `gallery-data.json` будет скачан на ваш компьютер
- Используйте для:
  - Сохранения резервной копии
  - Обновления данных на сервере
  - Синхронизации между разными экземплярами

## Структура данных

Каждое изображение содержит следующие поля:

```json
{
  "id": 1,
  "src": "Images/Gallery/photo.jpg",
  "name": "Название события",
  "description": "Подробное описание",
  "date": "2025-01",
  "tags": ["тег1", "тег2"]
}
```

## Автоматическое сохранение

Все изменения автоматически сохраняются в:
1. **localStorage** (браузер) - для быстрого доступа
2. Обновляется внутренняя переменная `this.images`

## Синхронизация с галереей

Главная галерея (`gallery.html`) автоматически загружает данные из `data/gallery-data.json`. После редактирования в админ-панели:
- Обновите страницу галереи
- Новые изображения будут отображены автоматически

## Советы

### При добавлении нового изображения:
1. Сначала скопируйте фото в папку `Images/Gallery/`
2. Укажите точный путь в админ-панели
3. Добавьте описательное название и теги для лучшей поисковости

### При редактировании:
- Используйте понятные названия (например: "Олимпиада 2025" вместо "IMG123")
- Добавляйте подробные описания для контекста
- Используйте релевантные теги для организации

### Дата в формате YYYY-MM:
- Примеры: `2025-01`, `2024-12`, `2023-06`
- Помогает отсортировать события по времени

## Решение проблем

### Изображения не загружаются
- Проверьте, что путь к файлу правильный
- Убедитесь, что файл существует в папке `Images/Gallery/`
- Проверьте расширение файла (jpg, png, gif, webp)

### Данные не сохраняются
- Проверьте, что заполнены все обязательные поля (путь и название)
- Попробуйте обновить страницу браузера
- Проверьте консоль браузера на ошибки (F12)

### При экспорте JSON
- Убедитесь, что браузер позволяет скачивать файлы
- Проверьте папку "Загрузки" / "Downloads"
- Используйте экспортированный файл для резервной копии
