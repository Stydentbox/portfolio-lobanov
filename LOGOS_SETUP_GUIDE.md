# Гайд по добавлению логотипов учебных заведений

## Структура папки для логотипов

Все логотипы должны находиться в папке: `Images/Logos/`

```
Images/
└── Logos/
    ├── adu-logo.png                    # Азербайджанский государственный университет
    ├── slav-uni-logo.png               # Славянский университет Республики Молдова
    ├── spbau-logo.png                  # Санкт-Петербургский Академический университет РАН
    ├── spbu-logo.png                   # Санкт-Петербургский государственный университет
    ├── dvfu-logo.png                   # Дальневосточный федеральный университет
    ├── mipt-logo.png                   # Московский Физико-технический институт
    └── uspe-logo.png                   # USPE Constantin Stere
```

## Как добавить логотип

### 1. Подготовка изображения

- **Формат**: PNG, JPG, GIF, WebP
- **Размер**: рекомендуется 200x200px или больше
- **Фон**: прозрачный или белый (лучше всего с прозрачностью)
- **Качество**: максимально четкое изображение

### 2. Сохранение логотипа

1. Поместите изображение логотипа в папку `Images/Logos/`
2. Дайте файлу имя согласно таблице выше
3. Используйте только латинские буквы в названии файла

### 3. Проверка

После добавления логотипа:
1. Откройте страницу `education.html` в браузере
2. Логотип должен появиться слева от названия учебного заведения
3. Если логотип не загружается, будет показана emoji-иконка

## Интеграция в HTML

Каждый логотип интегрирован следующим образом:

```html
<div class="education-logo-container">
    <img src="Images/Logos/FILENAME.png" alt="ALT_TEXT" class="education-logo" onerror="this.style.display='none'">
    <span class="education-logo-emoji" style="display:none;">EMOJI</span>
</div>
```

### Параметры:

- **src**: Путь к файлу логотипа
- **alt**: Альтернативный текст для доступности
- **onerror**: Скрывает изображение, если оно не найдено
- **education-logo-emoji**: Emoji, показываемое, если изображение недоступно

## Добавление новых логотипов для других учебных заведений

### Шаг 1: Добавить логотип в папку
```
Images/Logos/your-uni-logo.png
```

### Шаг 2: Обновить HTML в education.html
Найдите нужный раздел и добавьте:

```html
<div class="education-layout">
    <div class="education-logo-container">
        <img src="Images/Logos/your-uni-logo.png" alt="Your University Name" class="education-logo" onerror="this.style.display='none'">
        <span class="education-logo-emoji" style="display:none;">🏫</span>
    </div>
    <div class="education-date">DATES</div>
    <div class="education-content">
        <!-- Остальная информация об образовании -->
    </div>
</div>
```

### Шаг 3: Выбор подходящей emoji

Если логотип недоступен, используйте одну из этих emoji:
- 🏫 — Школа/Университет
- 🎓 — Образование
- 🇦🇿 — Флаг страны (Азербайджан)
- 🇷🇺 — Флаг страны (Россия)
- 🇲🇩 — Флаг страны (Молдова)
- 📚 — Книги
- 🎯 — Цель/Фокус

## CSS Классы

### `.education-logo-container`
- Контейнер для логотипа размером 100x100px
- На планшетах: 80x80px
- На мобильных: адаптивный

### `.education-logo`
- Изображение логотипа
- Масштабируется для заполнения контейнера
- Сохраняет пропорции (object-fit: contain)

### `.education-layout`
- Основная сетка образовательной записи
- 2 колонки: логотип (100px) + содержание
- На мобильных: 1 колонка (стёк)

## Оптимизация логотипов

### Сжатие изображений

Для уменьшения размера файлов используйте онлайн-инструменты:
- TinyPNG (tinypng.com)
- Compressor.io
- Optimizilla

### Рекомендуемые размеры файлов:
- Логотип 200x200px: 10-50 KB
- Логотип 400x400px: 30-100 KB

## Адаптивность

Макет автоматически адаптируется к разным размерам экранов:

- **Desktop (>768px)**: Логотип 100x100px слева, содержание справа
- **Планшет (768px-480px)**: Логотип 80x80px слева, меньший шрифт
- **Мобильный (<480px)**: Логотипы по центру, содержание снизу (стёк)

## Решение проблем

### Логотип не отображается

1. **Проверьте путь к файлу**
   - Папка: `Images/Logos/`
   - Расширение: .png, .jpg, .gif, .webp

2. **Проверьте размер файла**
   - Максимум 1 MB на файл
   - Рекомендуется до 100 KB

3. **Проверьте формат изображения**
   - Убедитесь, что это валидный PNG/JPG/GIF
   - Можно открыть в любом графическом редакторе

4. **Очистите кэш браузера**
   - Нажмите Ctrl+Shift+Delete
   - Удалите кэш
   - Перезагрузите страницу

### Логотип выглядит размытым

- Используйте изображение большего размера (400x400px или больше)
- Проверьте качество исходного файла
- Попробуйте другой формат (PNG вместо JPG)

### Логотип обрезается или искажается

- Убедитесь, что логотип имеет прозрачный фон
- Проверьте, что изображение квадратное (1:1)
- Используйте `object-fit: contain` (уже применено)
