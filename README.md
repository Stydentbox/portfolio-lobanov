# Портфолио Лобанова А.Н.

Статический сайт-портфолио преподавателя информатики с темной темой в ГОСТ-стиле.

## Структура проекта

```
Portfolio/
├── index.html              # Главная страница
├── achievements.html       # Достижения учеников  
├── education.html         # Образование и квалификация
├── publications.html      # Научные публикации
├── contact.html          # Контакты и форма связи
├── css/
│   └── style.css         # Единый стиль с темной темой
├── Images/
│   ├── Lobanov_AN.png    # Фотография профиля
│   ├── favicon.ico       # Иконки сайта
│   ├── favicon-16x16.png
│   ├── favicon-32x32.png
│   ├── apple-touch-icon.png
│   ├── android-chrome-192x192.png
│   ├── android-chrome-512x512.png
│   └── site.webmanifest
├── README.md
└── CLAUDE.md            # Инструкции для разработки
```

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (Grid, Flexbox, медиа-запросы)
- Минимальный JavaScript (навигация, кнопка "наверх")
- Адаптивный дизайн
- Темная тема (ГОСТ-стиль)
- Поддержка всех устройств

## Особенности

## Развертывание на GitHub Pages

1. Создайте новый репозиторий на GitHub
2. Загрузите все файлы проекта
3. В настройках репозитория включите GitHub Pages
4. Выберите источник: Deploy from a branch → main
5. Сайт будет доступен по адресу: `https://username.github.io/repository-name`
