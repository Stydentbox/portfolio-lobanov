# Инструкция по использованию админ-панели портфолио

## Обзор

Админ-панель предназначена для локального управления контентом портфолио без необходимости редактирования кода. Все данные хранятся в localStorage браузера.

## Доступ к админ-панели

Откройте файл `admin.html` в браузере:
- Локально: `file:///путь/к/проекту/admin.html`
- Или через локальный сервер: `http://localhost:порт/admin.html`

## Основные возможности

### 📷 Галерея

**Добавление фотографий:**

1. **Drag & Drop (Перетаскивание):**
   - Перетащите одно или несколько изображений в зону загрузки
   - Поддерживаемые форматы: JPG, PNG, GIF
   - Максимальный размер: 5 МБ

2. **Выбор файла:**
   - Кликните на зону загрузки
   - Выберите файл через диалоговое окно

3. **Заполнение информации:**
   - Название события
   - Описание (опционально)
   - Дата в формате ГГГГ-ММ (например: 2025-01)
   - Теги через запятую (например: событие, проект, ИИ)

4. **Важно:**
   - После перетаскивания файла система автоматически создаст путь `Images/Gallery/имя_файла.jpg`
   - **Обязательно скопируйте файл изображения в папку `Images/Gallery/`** вручную
   - После этого нажмите "Добавить в галерею"

**Управление фотографиями:**
- Просмотр всех добавленных фотографий в виде сетки
- Удаление фотографий кнопкой "Удалить"

**Экспорт/Импорт:**
- **Экспортировать JSON** - сохранить данные галереи в файл
- **Импортировать JSON** - загрузить данные из файла

---

### 🏆 Достижения

**Добавление достижения:**

1. Нажмите "+ Добавить достижение"
2. Заполните форму:
   - Название достижения
   - Год
   - Результат (например: "1 место в мировом рейтинге")
   - Описание
   - Уровень (Международный/Республиканский/Региональный)
   - Категория (Олимпиада/Конкурс/ИИ/Программирование/Другое)
   - Теги (через запятую)
   - Путь к изображению (опционально)

3. Нажмите "Сохранить достижение"

**Пример заполнения:**
```
Название: Intel® AI Global Impact Festival
Год: 2025
Результат: 1 место в мировом рейтинге
Описание: Республика Молдова заняла первое место...
Уровень: Международный
Категория: Искусственный интеллект
Теги: ИИ, международный, победа
Изображение: Images/achievements/intel_2025.jpg
```

### 📚 Публикации

**Добавление публикации:**

1. Нажмите "+ Добавить публикацию"
2. Заполните форму:
   - Авторы (например: Лобанов А. Н.)
   - Название публикации
   - Источник публикации (конференция, журнал)
   - Год публикации
   - Страницы (опционально)
   - Тип публикации (Статья/Книга/Диссертация/Материалы конференции)

3. Нажмите "Сохранить публикацию"

**Пример заполнения:**
```
Авторы: Лобанов А. Н.
Название: Информационные технологии в гибридном образовании
Источник: Материалы VIII международной научно-практической конференции
Год: 2018
Страницы: С. 191
Тип: Материалы конференции
```

---

### 🎓 Повышение квалификации

**Добавление курса:**

1. Нажмите "+ Добавить курс"
2. Заполните форму:
   - Название учебного заведения
   - Название программы
   - Дата начала (выбор из календаря)
   - Дата окончания (выбор из календаря)
   - Количество часов
   - Город
   - Тип документа (Удостоверение/Диплом/Другое)
   - Описание (опционально)
   - Теги/специализация (через запятую)

3. Нажмите "Сохранить курс"

**Пример заполнения:**
```
Учебное заведение: Московский Физико-технический институт
Программа: Машинное обучение и управление проектами в IT
Дата начала: 10.06.2020
Дата окончания: 24.08.2020
Часы: 72
Город: Москва
Тип документа: Удостоверение о повышении квалификации
Теги: Машинное обучение, Управление проектами
```

## Хранение данных

### LocalStorage

Все данные сохраняются в localStorage браузера под следующими ключами:
- `gallery` - данные галереи
- `achievements` - данные достижений
- `publications` - данные публикаций
- `education` - данные образования

### Резервное копирование

**ВАЖНО:** Данные хранятся только в браузере. Регулярно создавайте резервные копии!

**Экспорт данных:**
1. Перейдите на нужную вкладку
2. Нажмите "Экспортировать JSON"
3. Сохраните файл на компьютере

**Импорт данных:**
1. Нажмите "Импортировать JSON"
2. Выберите ранее сохраненный JSON-файл
3. Данные будут загружены в систему

## Интеграция с основным сайтом

### Для галереи

После добавления фотографий в админ-панели:

1. Экспортируйте данные галереи (кнопка "Экспортировать JSON")
2. Сохраните файл как `gallery-data.json`
3. Скопируйте его в папку `data/`
4. Убедитесь, что все изображения скопированы в `Images/Gallery/`
5. Обновите страницу `gallery.html` - новые фото появятся автоматически

### Для достижений, публикаций, образования

Данные из админ-панели пока хранятся только в localStorage. Для интеграции с HTML-страницами:

**Вариант 1: Ручное обновление**
- Экспортируйте JSON
- Вручную добавьте новые записи в соответствующие HTML-файлы

**Вариант 2: Динамическая загрузка (требует доработки)**
- Создайте JS-скрипты для загрузки данных из JSON
- Модифицируйте HTML-страницы для использования этих скриптов

## Решение проблем

### Изображения не отображаются в галерее

1. Проверьте, что файл изображения скопирован в `Images/Gallery/`
2. Убедитесь, что путь в данных совпадает с реальным расположением файла
3. Проверьте права доступа к файлу

### Данные не сохраняются

1. Убедитесь, что localStorage не отключен в браузере
2. Проверьте, что не используется режим инкогнито
3. Очистите кэш браузера и попробуйте снова

### Пропали данные

1. Проверьте, не очищали ли вы данные браузера
2. Восстановите из резервной копии (JSON-файл)
3. Если резервной копии нет - данные утеряны (важность регулярного экспорта!)

### Не работает drag & drop

1. Убедитесь, что используете современный браузер (Chrome, Firefox, Edge)
2. Проверьте, что файл является изображением
3. Убедитесь, что размер файла не превышает 5 МБ

## Рекомендации по работе

1. **Регулярно экспортируйте данные** - создавайте резервные копии минимум раз в неделю
2. **Организуйте файлы** - храните изображения в соответствующих подпапках
3. **Используйте понятные имена** - для файлов изображений используйте осмысленные названия
4. **Заполняйте все поля** - чем больше информации, тем лучше для SEO и пользователей
5. **Проверяйте результат** - после добавления контента проверьте его отображение на основном сайте
